# Запуск:
1. Клонируйте репозиторий

```shell
git clone https://github.com/levchig737/testXlsx.git && cd testXlsx
```

2. Скомпилируйте проект с помощью Maven

```shell
mvn clean compile
```

3. Запустите приложение

```shell
mvn spring-boot:run
```

## Запросы
Доступ по ссылке: http://localhost:8080/
